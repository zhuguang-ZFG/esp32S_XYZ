# M4.10 voiceprint demo status

Date: 2026-05-15

Status: design-time voiceprint voice task demo is verifiable.

## Evidence Boundary

This status is local/design-time evidence only. It is not a completion declaration and does not replace real audio voiceprint comparison, production ASR `speaker_ref` evidence, child re-enrollment capture, production push delivery, or U8/U1 real-device execution evidence.

Implemented scope:

- DeviceServer carries matched voiceprint metadata into `constraints.voiceprint` when submitting `/internal/v1/voice_task`.
- The metadata contains only routing/audit fields: `matched`, `member_id`, `display_name`, `member_type`, `speaker_ref`, `reason`, and optional `reenroll_hint`.
- BusinessServer accepts the constraints, stores them in task `constraintsJson`, and keeps them on the projected dispatch request.
- Regression coverage verifies DeviceServer payload construction, internal controller forwarding, and AppV2 service persistence/dispatch preservation.
- DeviceServer now changes the voice prompt when a submitted voice task needs primary approval, when the speaker is unknown but policy allows queueing, or when a child voiceprint has a `reenroll_hint`.
- The mini program primary-approval card parses `constraints.voiceprint` and shows the matched speaker, unknown-speaker review reason, or child re-enrollment warning before approve/reject.

Still not complete:

- Real audio voiceprint comparison.
- Real ASR-side `speaker_ref` production.
- Full child voiceprint re-enrollment capture flow; local UI currently surfaces the warning only.
- Production push notification for parent secondary confirmation; local primary can review pending voice tasks in the device detail page.
- U8/U1 real-device end-to-end execution.

Validation:

- `tests/ci/test_xiaozhi_server_voiceprint_cache.py` covers voiceprint policy, payload constraints, primary-approval prompt text, unknown-speaker prompt text, and child re-enrollment prompt text.
- `tests/ci/test_manager_mobile_device_info.py` covers the mini program voiceprint approval card tokens.
