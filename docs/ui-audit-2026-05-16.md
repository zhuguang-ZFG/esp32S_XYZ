## §6 发现的问题

### P1: 硬编码英文文案（未走 i18n）

以下文案使用了硬编码英文，应改为 `t('key')` 调用：

| 文件 | 行号 | 硬编码文案 |
|------|------|-----------|
| device-detail/index.vue | 745 | "Pending voice approvals" |
| device-detail/index.vue | 751 | "Refresh" |
| device-detail/index.vue | 781 | "Approve" |
| device-detail/index.vue | 784 | "Reject" |
| device-detail/index.vue | 791 | "No pending voice approvals" |
| device-list/index.vue | 85 | "Pending transfers" |
| device-list/index.vue | 107 | "Accept" |

**影响**: 中文用户看到英文按钮，体验不一致。
**修复建议**: 在 zh_CN.ts 中补充对应 key，模板中改用 `t()` 调用。

### P2: cancelled 状态无专门视觉标识

当前 `cancelled` 落入默认灰色，与"未知状态"无法区分。
**修复建议**: 为 cancelled 添加独立颜色（如黄色/橙色）或图标（停止符号）。

### P3: Edge-A WSS 无自动重连

当前断线后需要用户手动点击"连接并订阅"。
**修复建议**: 添加指数退避自动重连（3s/6s/12s/30s），最大重试 5 次后停止并提示。
