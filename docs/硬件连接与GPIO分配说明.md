# DLC Motor Control P1 硬件连接与 GPIO 分配说明

基于原理图文件：`DLC_Motor_Control_P1_V1.0_260513SCH(1).pdf`

本文档用于整理本项目的外部接口接线方式、两颗 ESP32-S3 的角色分工，以及每个 GPIO 的连接去向。

## 1. 系统角色分工

本板使用两颗 `ESP32-S3-WROOM-1-N16R8`：

- `U1`：建议作为写字机/雕刻机运动控制 MCU
  - 已连接 4 路步进驱动
  - 已连接 4 路原点传感器
  - 已连接 3 路舵机
  - 已连接压力传感器 `HX711`
  - 已连接激光控制
- `U8`：建议作为“小智”/AI 交互 MCU
  - 已连接摄像头 DVP 总线
  - 已连接音频 I2S、音频 ADC、Codec、功放使能
  - 已连接 I2C
  - 已连接 SD 卡信号
  - 具备独立 USB 烧录口

这类分工与当前原理图连接关系完全一致，因此推荐：

- `U1` 运行写字机/雕刻机控制固件
- `U8` 运行小智、语音、视觉、交互相关固件

两颗 MCU 通过串口互联：

- `U1.IO10` -> `M_U1TXD` -> `U8.IO11`
- `U1.IO11` -> `M_U1RXD` <- `U8.IO10`

也就是：

- `U1` 发给 `U8`：`U1.IO10`
- `U8` 发给 `U1`：`U8.IO11`

## 2. 电源结构

主输入为 `12V_DC`。

板上主要电源如下：

- `12V`：主功率输入，步进驱动和激光负载直接使用
- `+3V3`：数字逻辑电源，供两颗 ESP32-S3、驱动逻辑等使用
- `AU_3V3`：音频相关 3.3V 电源域
- `+5V_A`：舵机、激光相关外围和部分模拟/功放电路使用
- `2V8`：摄像头电源，由 `+3V3` 经过 LDO 生成
- `USB5V_VCC`：USB 口供电，用于烧录/USB 相关

## 6. 外设逐项接线说明

### 6.1 步进电机接线

每个步进接口都是 4 线双极步进电机接口。

接线原则：

1. 一组线圈接 `OUT1A` 和 `OUT1B`
2. 另一组线圈接 `OUT2A` 和 `OUT2B`
3. 不要把不同线圈混接

各接口定义：

- 电机 1：`J5-3=OUT1A`, `J5-4=OUT1B`, `J5-2=OUT2A`, `J5-1=OUT2B`
- 电机 2：`J1-3=OUT1A`, `J1-4=OUT1B`, `J1-2=OUT2A`, `J1-1=OUT2B`
- 电机 3：`J3-3=OUT1A`, `J3-4=OUT1B`, `J3-2=OUT2A`, `J3-1=OUT2B`
- 电机 4：`J4-3=OUT1A`, `J4-4=OUT1B`, `J4-2=OUT2A`, `J4-1=OUT2B`

如果接完方向与软件定义相反：

- 优先在固件里修改方向极性
- 或对调同一组线圈的两根线

### 6.2 原点传感器接线

原点传感器接口面向 12V 三线传感器，输入先进入光耦，再转换为 `U1` 可读的 `3.3V` 逻辑。

标准接法：

- `12V` -> 传感器电源正
- `DGND` -> 传感器地
- `MTO_SENSORx_OUT` -> 传感器输出

接口示例：

- 原点 1：`J25-1=12V`, `J25-2=MTO_SENSOR1_OUT`, `J25-4=DGND`
- 原点 2：`J6-1=12V`, `J6-2=MTO_SENSOR2_OUT`, `J6-4=DGND`
- 原点 3：`J7-1=12V`, `J7-2=MTO_SENSOR3_OUT`, `J7-4=DGND`
- 原点 4：`J8-1=12V`, `J8-2=MTO_SENSOR4_OUT`, `J8-4=DGND`

原理图备注说明可通过装配位切换极性：

- 遮光输出高，入光输出低
- 遮光输出低，入光输出高

所以选型和装配时要统一传感器输出逻辑。

### 6.3 舵机接线

每路舵机为标准 3Pin 接法：

- `Pin1 = DGND`
- `Pin2 = PWM`
- `Pin3 = +5V_A`

对应为：

- 舵机 1：`DGND`, `SERVO_PWM1`, `+5V_A`
- 舵机 2：`DGND`, `SERVO_PWM2`, `+5V_A`
- 舵机 3：`DGND`, `SERVO_PWM3`, `+5V_A`

注意：

- 舵机供电是 `+5V_A`
- 不能直接按 `3.3V` 舵机逻辑理解供电端

### 6.4 压力传感器接线

`J10` 对应 `HX711` 前端，适合桥式压力/称重传感器。

接线关系：

- `J10-1 = SENSOR_VDD`：传感器激励正
- `J10-2 = DGND`：传感器地
- `J10-3 = SENSOR_S-`：差分负信号
- `J10-4 = SENSOR_S+`：差分正信号

若外部传感器标记为 `E+`, `E-`, `A+`, `A-`，通常可按下列方式连接：

- `E+` -> `SENSOR_VDD`
- `E-` -> `DGND`
- `A-` -> `SENSOR_S-`
- `A+` -> `SENSOR_S+`

`HX711` 连接到 `U1`：

- `HX711_SCLK` -> `U1.IO7`
- `HX711_MISO` -> `U1.IO15`

原理图中速率说明为：

- `0 = 10Hz`
- `1 = 80Hz`

### 6.5 激光接线

激光接口 `J12` 是一个 12V 负载接口，由 `Q5(IRFZ44N)` 做低边开关控制。

接线方式：

- `J12-1 = 12V` -> 激光模块正端
- `J12-2 = SWITCHED_GND` -> 激光模块负端

控制关系：

- `U1.IO45` 输出 `LASER_CONTROL`
- `LASER_CONTROL` 驱动 `Q5`
- `Q5` 决定 `J12-2` 是否接地导通

也就是说：

- 激光负载电流不直接经过 MCU
- MCU 只负责控制 MOS 管开关
