# HR4988E 细分与电流配置核对

**日期**: 2026-05-16
**核对源**: 原理图 PDF (`DLC_Motor_Control_P1_V1.0_260513SCH.pdf` 第 4 页)、PADS 网表 (`*SIGNAL*` 段)、HR4988 datasheet
**核对目标**: `firmware/u1-grbl/.../Machines/dlc_motor_control_p1.h`

## 1. 微步细分配置

### 1.1 硬件连接

MS1/MS2/MS3 三脚通过 R10 (1K) 上拉到 +3V3，形成同网连接：

```
+3V3 → R10 (1K) → MS1 = MS2 = MS3 = HIGH
```

PADS 网表证据链：
- `U52.9 (MS1) → @@@D28 → R10 → @@@O14 (+3V3)`
- `U52.10 (MS2) → @@@D29 → @@@D28` (同网)
- `U52.11 (MS3) → @@@D30 → @@@D29` (同网)

四路驱动 (U52/U2/U3/U4) 配置相同。

### 1.2 微步真值表

HR4988E 与 A4988 引脚兼容，微步真值表相同：

| MS1 | MS2 | MS3 | 细分 |
|-----|-----|-----|------|
| L   | L   | L   | 全步 |
| H   | L   | L   | 1/2  |
| L   | H   | L   | 1/4  |
| H   | H   | L   | 1/8  |
| H   | H   | H   | 1/16 |

**本板配置**: MS1=H, MS2=H, MS3=H → **1/16 微步**

### 1.3 原理图标注勘误

原理图 PDF 第 4 页顶部标注 `1/32`，这是 TMC2208 时代的遗留标注（TMC2208 在 MS1=MS2=H 时为 1/32 微步）。实际芯片 HR4988E 最大仅支持 1/16 微步。该标注为历史错误，不影响硬件功能。

### 1.4 固件 steps/mm 验算

| 轴 | 固件值 | 反推传动比 | 对应机构 | 判定 |
|----|--------|-----------|----------|------|
| X  | 80.0 steps/mm | 200×16÷80 = 40mm/圈 | GT2 同步带 20齿×2mm | 合理 |
| Y  | 80.0 steps/mm | 200×16÷80 = 40mm/圈 | GT2 同步带 20齿×2mm | 合理 |
| Z  | 400.0 steps/mm | 200×16÷400 = 8mm/圈 | T8 丝杆 4头×2mm螺距 | 合理 |

固件配置与 1/16 微步 + 常见传动机构完全匹配。

## 2. 电流限制配置

### 2.1 VREF 分压电路

每路驱动的 REF 引脚 (pin 17) 由电阻分压设定：

```
+3V3 ─── R_high (9.1K) ─┬─ REF (pin 17)
                         │
                    R_low (2.2K)
                         │
                        GND
```

各通道对应电阻：

| 通道 | R_high | R_low | 位号 |
|------|--------|-------|------|
| U52 (X轴) | R14 = 9.1K | R15 = 2.2K | J5 |
| U2 (Y轴)  | R30 = 9.1K | R31 = 2.2K | J1 |
| U3 (Y2轴) | R43 = 9.1K | R44 = 2.2K | J3 |
| U4 (Z轴)  | R56 = 9.1K | R57 = 2.2K | J4 |

### 2.2 电流计算

```
VREF = 3.3V × 2.2K / (9.1K + 2.2K) = 0.642V

ITripMAX = VREF / (8 × Rs) = 0.642 / (8 × 0.1) = 0.803A/相

VSENSE = ITripMAX × Rs = 0.803 × 0.1 = 0.080V
```

### 2.3 SENSE 电阻

每路驱动有两个 SENSE 电阻（SENSE1/SENSE2），均为 0.1 ohm：

| 通道 | SENSE1 | SENSE2 |
|------|--------|--------|
| U52 | R83 = 0.1R | R16 = 0.1R |
| U2  | R32 = 0.1R | R33 = 0.1R |
| U3  | R45 = 0.1R | R46 = 0.1R |
| U4  | R58 = 0.1R | R59 = 0.1R |

### 2.4 安全裕度

| 参数 | 计算值 | datasheet 限制 | 裕度 |
|------|--------|---------------|------|
| VSENSE | 0.080V | < 0.5V | 6.25x |
| VBB (电机电源) | 12V | 8~38V | 范围内 |
| VDD (逻辑电源) | 3.3V | 3.0~5.5V | 范围内 |

## 3. STEP/DIR 时序配置

### 3.1 固件时序参数

```c
#define DEFAULT_STEP_PULSE_MICROSECONDS   4    // STEP 脉宽 4us
#define STEP_PULSE_DELAY                  1    // DIR→STEP 建立时间 1us
```

### 3.2 与 datasheet 对比

| 参数 | datasheet 最小值 | 固件配置 | 裕度 |
|------|-----------------|----------|------|
| STEP 高电平最小宽度 | 1us | 4us | 4x |
| STEP 低电平最小宽度 | 1us | 由步进频率决定 | - |
| DIR 建立时间 | 200ns | 1us | 5x |

### 3.3 串联电阻影响

STEP/DIR 信号经 1K 串联电阻到 HR4988E 输入端。考虑输入电容约 5pF：

```
RC 时间常数 = 1K × 5pF = 5ns
上升时间 (10%~90%) ≈ 2.2 × RC = 11ns
```

对 4us 脉宽影响可忽略。

## 4. 使能控制

### 4.1 ENABLE# 电路

```
U1.IO4 (MOTOR_EN) ─── R_series (1K) ─┬─ ENABLE# (pin 2)
                                       │
                                  R_pullup (9.1K)
                                       │
                                      +3V3
```

- ENABLE# 低有效：低电平使能输出，高电平关断
- 上电默认：9.1K 上拉到 +3V3 → ENABLE#=HIGH → 输出关断（安全）
- 固件 `DEFAULT_INVERT_ST_ENABLE=0`：IO4 输出 LOW 时使能电机

### 4.2 RESET# / SLEEP#

RESET# 和 SLEEP# 与 MS1/MS2/MS3 同网，经上拉到 +3V3：
- RESET# = HIGH → 不复位
- SLEEP# = HIGH → 不休眠

## 5. 已知风险项

| 编号 | 项目 | 等级 | 说明 | 固件影响 |
|------|------|------|------|----------|
| R-007 | ROSC=1K | L2 | tOFF≈1.2us，偏离推荐~30us，开关频率偏高 | 无需改固件，需实测温升 |
| R-008 | CP=22nF | L2 | 低于推荐 0.1uF/50V，高负载时可能影响高侧驱动 | 无需改固件，需实测 |

## 6. 总结

| 检查项 | 原始资料 | 固件配置 | 一致性 |
|--------|----------|----------|--------|
| 微步细分 | MS1=MS2=MS3=H → 1/16 | steps/mm 按 1/16 标定 | 一致 |
| 电流限制 | VREF=0.642V, Rs=0.1R → 0.80A | 无固件可调参数 | 硬件决定 |
| STEP 脉宽 | datasheet min 1us | 4us | 裕度充足 |
| DIR 建立时间 | datasheet min 200ns | 1us | 裕度充足 |
