# AI 提示词改造计划

**基于**: system-prompts-and-models-of-ai-tools 项目学习
**目标**: 优化写字机 AI 简笔画生成的 prompt 质量和准确性

## 1. 当前问题

当前 `qwen-image-2.0` 的 prompt 只有一句话：
```
{用户输入}，简笔画，黑色线条，纯白背景，无阴影无填充无文字
```

问题：
- 没有角色定义，模型不知道自己在为什么场景生图
- 没有约束条件（线条粗细、复杂度、构图）
- 没有输出格式要求（导致有时生成复杂渐变图）
- 没有针对矢量化友好的引导

## 2. 学到的关键模式

### 2.1 来自 Kiro 的模式分类器
- 根据用户输入的复杂度，选择不同的生成策略
- 简单图形（星星、心形）→ 预置素材库（0ms）
- 中等图形（猫、花）→ AI 生图 + 矢量化（3.5s）
- 复杂场景（风景、建筑）→ 拒绝或简化提示

### 2.2 来自 Cursor 的分层约束
- 明确的能力边界声明
- 工具调用纪律（最小化、批量化）
- 语义搜索优先（先找相似素材）

### 2.3 来自 Windsurf 的记忆系统
- 记住用户偏好（线条风格、复杂度偏好）
- 跨会话保持一致性

### 2.4 来自 Devin 的规划模式
- 先收集信息，再执行
- 明确的 planning → execution 分离

## 3. 改造方案

### 3.1 图像生成 Prompt 模板（核心改进）

**System Prompt（角色定义 + 约束）**:
```
你是一台笔绘机的视觉设计师。你的作品将被一支笔在纸上一笔一笔画出来。

设计原则：
- 只用黑色线条，不要灰度、阴影、渐变或填充色块
- 线条清晰连续，粗细均匀（约2-3px）
- 构图居中，留白适当，主体占画面60-80%
- 简笔画风格：用最少的线条表达最多的特征
- 避免密集交叉线、细碎纹理、文字
- 背景纯白，无任何装饰元素

技术约束（为矢量化优化）：
- 线条之间保持间距（至少5px），避免粘连
- 封闭图形的线条要完全闭合
- 避免极细的线条（<1px）和极粗的色块
- 图形轮廓优先，内部细节从简
```

**User Prompt 模板**:
```
画一个{主题}的简笔画。
风格：{简约/可爱/写实轮廓}
复杂度：{低/中/高}（对应约10/20/40笔画）
```

### 3.2 模式分类器（路由策略）

在服务端实现请求分类：

```java
public enum DrawMode {
    PRESET,      // 预置素材命中 → 0ms
    AI_SIMPLE,   // 简单图形 → 快速生成
    AI_DETAILED, // 复杂图形 → 高质量生成
    REJECTED     // 超出能力范围 → 友好拒绝
}
```

分类规则：
| 输入特征 | 路由 | 示例 |
|----------|------|------|
| 匹配预置关键词 | PRESET | "星星"、"心形"、"太阳" |
| 单一具体物体 | AI_SIMPLE | "猫"、"花"、"鱼" |
| 带修饰的物体 | AI_DETAILED | "戴帽子的猫"、"开花的树" |
| 抽象/场景/多物体 | REJECTED | "春天的公园"、"城市夜景" |

### 3.3 Prompt 变体（按模式）

**AI_SIMPLE 模式**:
```
{主题}的简笔画，极简风格，5-15条线，黑色线条，纯白背景，
线条清晰不交叉，适合笔绘机绘制
```

**AI_DETAILED 模式**:
```
{主题}的简笔画，清晰轮廓风格，15-30条线，黑色线条，纯白背景，
线条间距适当不粘连，主体居中占画面70%，无文字无阴影
```

### 3.4 质量反馈循环

参考 Cursor 的"根因分析"模式：
- 矢量化后检测路径质量（点数、段数、覆盖率）
- 如果质量不达标，自动调整 prompt 重试（最多 2 次）
- 记录哪些 prompt 变体效果好，逐步优化

```java
if (strokeCount > 50 || coverageRatio < 0.3) {
    // 图太复杂或太空，调整 prompt 重试
    prompt = simplifyPrompt(originalPrompt);
    retry();
}
```

### 3.5 用户偏好记忆

参考 Windsurf 的记忆系统：
```yaml
user_preferences:
  style: cute          # 可爱/简约/写实
  complexity: medium   # 低/中/高
  line_weight: normal  # 细/正常/粗
  favorites: [猫, 花, 星星]  # 常画主题
```
